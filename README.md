# Telegram Workflow Notification Bot

Bot quản lý và gửi thông báo tự động cho nhiều Group/Channel. Owner thiết lập Workflow gồm nhiều bước (text, ảnh, video, sticker, delay, copy/forward message...), bot sẽ thực hiện tuần tự.

## Yêu cầu
- Node.js >= 16
- npm
- Tài khoản Telegram, đã tạo Bot qua [@BotFather](https://t.me/BotFather)

## Cài đặt

1. **Clone hoặc tải source code**
2. **Cài dependencies**
   ```bash
   npm install
   ```
